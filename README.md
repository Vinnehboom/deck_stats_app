## General

An application intended to support Pokemon TCG players in keeping track of their winrates and matchup data.

The data and decks are intended to be shared with testing groups and with all users of the application if so desired.

Add special notes to your records, keep track of the lines and gotchas for each matchup, and filter data based on whether you went first or second, won the coinflip,...
